About

PostgreSQL-compatible proxy translating queries to SQLite.

Metrics are available at /metrics in Prometheus format.

Metrics Preview

Key metrics exposed:

  • pgqt_requests_total - Total requests
  • pgqt_query_duration_seconds - Query latency histogram
  • pgqt_connections_active - Active connections
  • pgqt_queries_*_total - Query type breakdown

For full metrics, visit /metrics.
